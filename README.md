# 🧑‍💼  Employee Management System 

## 📌 Overview
The **Employee Management System ** is a **Spring Boot** application built with **Java, Thymeleaf, Bootstrap, and MySQL**.  
It provides an intuitive interface to manage employee data, including **adding, updating, deleting, filtering, and sorting** employees.  
The application ensures a **scalable backend**, **responsive design**, and **dynamic data handling** for smooth user experience.

---

## 🚀 Features
- ➕ **Add Employee** – Create new employee records.
- ✏ **Update Employee** – Modify existing employee details.
- ❌ **Delete Employee** – Remove employees from the system.
- 🔍 **Filter by Department** – Search employees based on department.
- 📊 **Sort by Salary** – Sort employee list in ascending/descending order.
- 📱 **Responsive Design** – Works seamlessly on desktop and mobile.
- 🗄 **MySQL Database** – Stores employee information securely.

---

## 🛠 Tech Stack
- **Backend**: Spring Boot (MVC Architecture), Java 17+
- **Frontend**: Thymeleaf, Bootstrap 5, HTML5, CSS3
- **Database**: MySQL
- **Build Tool**: Maven
- **Version Control**: Git & GitHub
